### 🥩 番茄燴牛肉

**食材**（2.5人份）：
- 主料：牛肉 1塊（約150g）、蕃茄 2個（約250g）、胡蘿蔔 2根（約200g）
- 配料：薑 3片、蒜 2瓣（切片）、蔥 1段（切段）
- 調味料：醬油 8ml（約等於鹽 1.1g）、鹽 0.3g、油 10ml（無不粘鍋多2ml）

**食材處理**：
1. 牛肉逆紋切薄片，用廚房紙吸乾血水
2. 牛肉片加醬油3ml（預留5ml）、薑汁（薑1片磨蓉擠汁），拌勻醃10分鐘
3. 蕃茄洗淨，去蒂，切小塊；胡蘿蔔去皮，切滾刀塊（約2cm）

**烹飪步驟**：
1. 鍋燒熱 → 放油6ml → 油熱後放牛肉片 → 大火快速滑炒至變色（約40秒）→ 立刻盛出
2. 鍋中放油4ml → 爆香蒜片、薑2片 → 放蕃茄塊 → 中火炒2-3分鐘至軟化出湯汁
3. 放胡蘿蔔塊 → 翻炒1分鐘 → 加熱水150ml、醬油5ml → 加蓋中火燜 **8分鐘** 至胡蘿蔔軟熟
4. 開蓋 → 倒回牛肉 → 大火收汁1-2分鐘 → 加鹽0.3g → 撒蔥段 → 出鍋

**⏱️ 烹飪時間**：約 15 分鐘

### 🍳 叉燒炒蛋

**食材**（2.5人份）：
- 主料：叉燒 約150g（昨晚剩的）、雞蛋 3個
- 配料：蔥 1段（切蔥花）
- 調味料：鹽 0.2g（⚠️ 叉燒已有味，鹽比平時少！）、油 8ml（無不粘鍋多2ml）

**食材處理**：
1. 叉燒切細粒（約1cm方丁）；雞蛋打散，加鹽0.2g攪勻

**烹飪步驟**：
1. 鍋燒熱 → 放油5ml → 油熱後倒入蛋液 → 快速劃散至半凝固（約20秒）
2. 撒入叉燒粒 → 翻炒均勻至蛋全熟（約30秒）
3. 撒蔥花 → 出鍋

**⏱️ 烹飪時間**：約 5 分鐘

### 🍲 枝竹火腩冬菇煲

**食材**（2.5人份）：
- 主料：燒肉（火腩）約200g、枝竹 2-3條（已泡發）、大冬菇 3朵（已泡發）
- 配料：胡蘿蔔 1根（約100g，切片）、薑 3片、蒜 3瓣（拍扁）、蔥 1段（切段）
- 調味料：史雲生雞湯 250ml（1盒）、醬油 **3ml**（約等於鹽 0.4g ⚠️ 雞湯有鹽，比平時少）、鹽 **0g**（先不加，最後試味決定）、油 3ml（燒肉自帶油！）

**食材處理**：
1. 泡發大冬菇 → 去蒂 → 切厚片（大冬菇肉厚，切片更出味；泡菇水留80ml備用）
2. 泡發枝竹 → 瀝乾 → 切段（約5cm）
3. 燒肉切塊（約3cm，連皮）

**烹飪步驟**：
1. 鍋大火燒熱 → 放油3ml → 放燒肉塊 → 中火煸炒1-2分鐘至出油、邊角微焦
2. 放薑片、蒜瓣、冬菇 → 炒出香味（約1分鐘）
3. 放枝竹、胡蘿蔔片 → 翻炒均勻
4. 加雞湯250ml、泡菇水80ml、醬油3ml → 大火煮滾 → 轉中小火加蓋燜 **12-15分鐘** 至枝竹軟腍入味
5. 開蓋 → 先試味（雞湯+燒肉本身夠鹹就不用加鹽） → 大火收汁1-2分鐘至湯汁略濃
6. 撒蔥段 → 連鍋上桌（或轉砂鍋更保溫）

**⏱️ 烹飪時間**：約 20 分鐘

### 🥬 蒜蓉炒菜心

**食材**（2.5人份）：
- 主料：菜心 半斤（約250g）
- 配料：蒜 2瓣（切片）
- 調味料：鹽 0.8g、油 8ml（水油炒法，無不粘鍋多2ml）

**食材處理**：
1. 菜心逐棵洗淨（流動水沖洗30秒以上），瀝乾，摘去老莖

**烹飪步驟**：
1. 鍋燒熱 → 放油4ml → 爆香蒜片
2. 放菜心莖部先炒30秒 → 再放葉部 → 大火快炒
3. 淋熱水30ml → 加蓋燜30秒
4. 加鹽0.8g、淋剩餘4ml油 → 炒勻出鍋

**⏱️ 烹飪時間**：約 5 分鐘

## ⚠️ 安全提示

- ✅ 牛肉全熟（回鍋燴煮至全熟 well-done）
