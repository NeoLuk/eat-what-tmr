# 少盐少油少糖烹饪实操指南

## 少盐策略

### 目标
- 普通孕晚期：每人每餐盐 ≤ 3g（全天 ≤ 6g）
- 水肿/高血压：每人每餐盐 ≤ 1.5g（全天 ≤ 3g）

### 盐量换算表
| 调味料 | 用量 | 约等于盐量 |
|--------|------|-----------|
| 食盐 | 1g | 1g |
| 酱油（生抽/老抽） | 15ml（1汤匙） | 2g |
| 蚝油 | 15ml（1汤匙） | 1g |
| 豆瓣酱 | 15g | 1g |
| 甜面酱 | 15g | 0.5g |
| 鱼露 | 5ml | 1g |

**原则**：如果用了酱油/蚝油，要相应减少盐的用量。在菜谱中标注"盐 Xg + 酱油 Xml，总钠约等于 Xg 盐"。

### 减盐不减味的技巧

1. **天然鲜味替代**（不靠盐提鲜）：
   - 番茄（谷氨酸天然来源）
   - 香菇、口蘑（鸟苷酸）
   - 海带、紫菜
   - 虾皮（少量）
   - 自制无盐鸡汤/骨头汤

2. **香味增强**：
   - 葱、姜、蒜、香菜（新鲜爆香）
   - 花椒、胡椒（少量）
   - 八角、桂皮（炖肉时1-2颗即可）
   - 柠檬汁、醋（酸味可以降低对盐的感知）

3. **烹饪时机**：
   - 起锅前再放盐（盐在表面，味蕾直接感知）
   - 分次加盐不如最后一次性调整
   - 提前腌制肉时少放盐，用姜葱汁去腥

4. **西式低盐调味**：
   - 迷迭香、百里香、罗勒、莳萝
   - 蒜粉、洋葱粉（确认无盐添加）
   - 柠檬皮屑、橙皮屑
   - 黑胡椒
   - 无盐蔬菜高汤

### 每餐盐分检查清单
- 每个荤菜用盐 ≤ 1g（用了酱油则减至 ≤ 0.5g）
- 每个素菜用盐 ≤ 1g
- 汤品用盐 ≤ 1g（或用天然鲜味不放盐）
- 一餐总量：荤 + 素 + 汤 ≤ 3g 盐

## 少油策略

### 目标
- 每人每餐烹饪用油 ≤ 15ml（约1汤匙）
- 一餐中所有菜的用油总量

### 用油换算
| 油品 | 15ml（1汤匙） | 5ml（1茶匙） |
|------|--------------|-------------|
| 橄榄油 | ~120 kcal | ~40 kcal |
| 山茶油 | ~120 kcal | ~40 kcal |
| 亚麻籽油 | ~120 kcal | ~40 kcal |
| 菜籽油 | ~120 kcal | ~40 kcal |
| 芝麻油 | ~130 kcal（提香用，不大量用） | ~45 kcal |

### 烹饪方式优先级

| 优先级 | 方式 | 用油量 | 适用菜式 |
|--------|------|--------|----------|
| 🥇 最佳 | 蒸 | 0ml | 鱼、鸡肉、蛋羹、蔬菜 |
| 🥇 最佳 | 煮 | 0ml | 汤、粥、白灼菜 |
| 🥈 优秀 | 炖/焖 | 0-2ml | 肉类、豆制品、根茎蔬菜 |
| 🥈 优秀 | 水煎 | 2-3ml | 饺子、锅贴、煎蛋（水代替油） |
| 🥉 良好 | 快炒 | 5-8ml | 青菜、肉丝、蛋类（不粘锅+少量油） |
| 🥉 良好 | 烤箱烤 | 3-5ml（涂抹） | 三文鱼、鸡胸、蔬菜 |
| ⚠️ 偶尔 | 煎 | 8-10ml | 鱼排、鸡胸（不粘锅） |
| ❌ 禁止 | 油炸/深炸 | 大量 | — |

### 减油实操技巧

1. **锅具选择**：
   - 不粘锅：快炒只需3-5ml油
   - 蒸锅/蒸烤箱：零油烹饪
   - 空气炸锅：只需涂抹极少油或利用食材本身油脂

2. **烹饪手法**：
   - 热锅凉油：锅烧热再放油，减少吸油
   - 水炒法：锅中放水(50ml)+油(3ml)，水沸腾时下菜，水油混合减少用油量
   - 先煎后炖：煎肉时的油可以继续用于炖煮，不额外加油
   - 利用食材自身油脂：五花肉、鸡皮先干煸出油，用肉自身的油来炒菜

3. **调味替代**：
   - 凉拌菜用醋、柠檬汁、蒜泥代替大量香油
   - 沙拉酱用希腊酸奶 + 柠檬汁 + 黑胡椒代替（传统沙拉酱含油量极高）
   - 炒菜出锅前淋少量芝麻油（3-5滴）增加香气，不需大量炒油

4. **每餐用油分配**：
   - 如有汤（清汤无油）：炒菜可各用5-8ml
   - 如全炒菜：每人每菜用油 ≤ 5ml（2-3个菜总计 ≤ 15ml）
   - 有蒸菜+炒菜：蒸菜0油，炒菜可用8-10ml

### 鱼类烹饪方法（不要默认蒸鱼！）

用户可能不喜欢清蒸鱼。以下是孕期安全的鱼类替代做法，按健康程度排序：

| 做法 | 用油量 | 适合鱼种 | 要点 |
|------|--------|----------|------|
| 🥇 少油煎 | 5-8ml | 红衫鱼、马头鱼、黄花鱼、三文鱼 | 锅烧热→姜片擦锅→放油→中小火煎至两面金黄。**无不粘锅时**：锅一定要烧热，油温上来再放鱼，煎的时候不要频繁翻动。煎好后淋少许酱油+水焖1分钟即成简易红烧。 |
| 🥈 水煎鱼 | 3ml | 各种海鱼 | 锅中放水50ml+油3ml，水沸放鱼，加盖中火焖至水干，鱼底面微焦。口感接近煎但用油极少。 |
| 🥉 红烧（减盐版） | 5ml | 鲩鱼、石斑、鲫鱼、各种海鱼 | 少油煎香→加姜葱蒜→加水+少量酱油（5ml/人）+少许醋→焖煮至汤汁浓稠。**不放糖**——用洋葱或番茄的天然甜味代替。**不放料酒**——用姜汁去腥。 |
| 🏅 鱼汤 | 0-2ml | 鲫鱼、石斑鱼骨、三文鱼头、黄花鱼 | 鱼煎至微黄（2ml油）→加开水（一定要开水，汤才白）→加姜片、豆腐、蔬菜→大火滚10分钟→撇油→加盐调味。 |
| 🏅 烤箱纸包鱼 | 3ml | 三文鱼、鳕鱼、石斑鱼柳 | 鱼柳放烘焙纸上→淋橄榄油+柠檬汁+香草→纸包密封→180°C烤15-18分钟。全熟且鱼肉嫩滑。需要烤箱。 |

**香港/沿海地区常见海鱼及推荐做法**：
| 鱼种 | 推荐做法 | 特点 |
|------|----------|------|
| 红衫鱼 | 少油煎、水煎 | 肉质细嫩，香港街市最常见 |
| 马头鱼 | 少油煎、红烧 | 肉厚刺少，适合孕妇 |
| 黄花鱼 | 煎、红烧、清蒸 | 低汞安全，DHA丰富 |
| 石斑鱼 | 红烧、姜葱炒 | 高档海鱼，肉厚嫩滑 |
| 鲩鱼（草鱼） | 红烧、鱼汤 | 淡水鱼，香港常见 |
| 三文鱼 | 纸包烤、少油煎 | DHA极高，选养殖或太平洋野生 |
| 鳕鱼 | 纸包烤、蒸 | 选真鳕鱼，避免油鱼冒充 |

### 煎鱼不粘锅技巧（无不粘锅时）

1. **锅要烧热**：空锅大火烧至冒轻烟（约30-60秒）
2. **姜擦锅**：切一块姜，用断面在热锅底部反复擦拭（形成隔离膜）
3. **再放油**：擦完姜后再放油，油热后放鱼
4. **不要翻**：下锅后中小火煎2-3分钟，底面金黄再翻面，频繁翻动必粘
5. **鱼身擦干**：煎之前用厨房纸吸干鱼身水分（水分是粘锅元凶）
6. **撒少许盐在锅底**：油热后在锅底均匀撒一小撮盐，再放鱼（盐形成微隔离层）

### 每周汤品安排

用户如喜欢喝汤，每周可安排1-2次肉汤（孕期非必须，但满足饮食习惯）：

- **鱼汤**（首选）：鲫鱼豆腐汤、石斑鱼骨番茄汤、三文鱼头汤
- **鸡汤**：去皮鸡腿+红枣2颗+枸杞10粒+姜片，撇净浮油
- **排骨汤**：瘦排骨+海带/莲藕/冬瓜，焯水去血沫后再炖，撇油
- **汤品原则**：
  - 煲汤前肉类先焯水去血沫和部分脂肪
  - 炖好后**必须撇油**（用勺子或放凉后去表面凝固油脂）
  - 一锅汤总盐量 ≤ 2g（因为要分多人喝）
  - 不放任何中药材（除非医生明确许可）
  - 汤里的肉也要吃，营养不只在汤里
  - 喝1-2碗即可，不要当水喝（汤中的盐和嘌呤需要控制）

## 少糖策略

### 目标
- 添加糖 = 0g
- 天然糖（来自食材本身）无需刻意限制，但注意总量

### 禁止的糖类
| 名称 | 常见于 |
|------|--------|
| 白砂糖/白糖 | 炒菜、糖醋、烘焙 |
| 冰糖 | 红烧、糖水 |
| 红糖/黑糖 | 糖水、粥、养生茶 |
| 蜂蜜 | 饮品、甜点（孕期可少量食用，但严格控量+确保巴氏消毒） |
| 果葡糖浆 | 加工食品、饮料 |
| 各种糖浆 | 枫糖浆、麦芽糖浆、龙舌兰糖浆 |

### 天然甜味替代

当菜谱需要甜味时，使用以下天然食材：

| 替代食材 | 甜度来源 | 适用场景 | 用量参考 |
|----------|---------|----------|----------|
| 红枣 | 天然果糖 | 炖汤、粥品、蒸菜 | 2-3颗/人 |
| 枸杞 | 天然果糖 | 汤品、蒸鱼、炒菜 | 5-10粒/人 |
| 南瓜 | 天然糖分 | 主食替代、汤品、蒸菜 | 150-200g/人 |
| 胡萝卜 | 天然糖分 | 炒菜、炖汤、沙拉 | 50-100g/人 |
| 甜椒（红/黄） | 天然糖分 | 炒菜、沙拉、烤菜 | 半个/人 |
| 洋葱 | 天然糖分（炒香后释出） | 各种菜式 | 1/4-1/2个/人 |
| 番茄 | 天然果糖 | 炒菜、炖菜、汤品 | 1个/人 |
| 苹果 | 果糖 | 炖汤、沙拉、烤菜 | 半个/人 |
| 梨 | 果糖 | 炖汤 | 半个/人 |
| 玉米 | 天然糖分 | 主食、汤品、沙拉 | 半根/人 |

### 需要警惕的"隐藏糖"

以下常见调味料和食材含糖，使用时要留意并在菜谱中标明：
- **番茄酱/ketchup**：1汤匙约含4g糖 → 用新鲜番茄替代
- **沙拉酱/蛋黄酱**：含糖+高油 → 用酸奶+柠檬替代
- **甜面酱**：含糖+含盐 → 少量使用或不用
- **照烧酱**：含糖量很高 → 自制减糖版或用酱油+姜汁替代
- **市售酸奶**：可能含大量添加糖 → 选原味无糖酸奶
