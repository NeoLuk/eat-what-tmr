# 应季食材参考（中国/北半球）

## 使用说明

生成菜谱时优先使用当季食材——更新鲜、更便宜、更安全。如果用户所在地区气候特殊或南半球，根据当地情况调整。

当不确定某食材是否当季时，选择常见且容易买到的替代食材。

---

## 按月食材速查

### 一月（冬季）
**蔬菜**：大白菜、白萝卜、胡萝卜、土豆、山药、莲藕、冬笋、菠菜、芹菜、芥菜、南瓜
**海鲜**：带鱼、鲈鱼、鲫鱼、草鱼、黄花鱼
**水果**：橙子、柑橘、柚子、苹果、梨、猕猴桃
**适合的烹饪方式**：炖、煲、蒸、火锅式汤锅、煲汤

### 二月（冬末）
**蔬菜**：同1月 + 韭菜（初春头茬韭菜）、荠菜（早春野菜）
**海鲜**：同1月 + 河虾开始上市
**水果**：同1月
**适合的烹饪方式**：同1月

### 三月（初春）
**蔬菜**：春笋、韭菜、荠菜、菠菜、芦笋、香椿、豌豆苗、生菜、油菜
**海鲜**：河虾、鲈鱼、桂鱼、黄花鱼、蛏子、蛤蜊
**水果**：草莓、菠萝、芒果（早期）、枇杷
**适合的烹饪方式**：清炒、白灼、凉拌（注意清洗）、蒸

### 四月（春季）
**蔬菜**：芦笋、豌豆、蚕豆、荷兰豆、西芹、莴笋、茭白、空心菜、蒜苔
**海鲜**：鲈鱼、桂鱼、黄花鱼、鲳鱼、河虾、皮皮虾
**水果**：草莓、樱桃、芒果、桑葚
**适合的烹饪方式**：清炒、蒸、白灼、轻煮

### 五月（春末夏初）
**蔬菜**：黄瓜、番茄、茄子、青椒、四季豆、苦瓜、丝瓜、苋菜、空心菜
**海鲜**：鲈鱼、黄花鱼、带鱼、花蛤、蛏子
**水果**：樱桃、杨梅、荔枝、桃子、甜瓜、西瓜（早期）
**适合的烹饪方式**：凉拌（注意清洗）、快炒、蒸、清汤

### 六月（初夏）
**蔬菜**：番茄、黄瓜、茄子、苦瓜、冬瓜、丝瓜、豇豆、毛豆、秋葵
**海鲜**：鲈鱼、黄花鱼、多宝鱼、基围虾、花蛤、蛏子
**水果**：西瓜、桃子、李子、荔枝、杨梅、蓝莓
**适合的烹饪方式**：凉拌、快炒、蒸、清汤、沙拉（确保清洗干净）

### 七月（盛夏）
**蔬菜**：冬瓜、苦瓜、丝瓜、番茄、茄子、豇豆、秋葵、空心菜、苋菜
**海鲜**：鲈鱼、多宝鱼、基围虾、花蛤、蛏子、鱿鱼
**水果**：西瓜、桃子、葡萄、龙眼、蓝莓、哈密瓜
**适合的烹饪方式**：蒸、凉拌、清汤、沙拉、白灼

### 八月（盛夏）
**蔬菜**：冬瓜、苦瓜、丝瓜、番茄、南瓜、莲藕、芋头、山药
**海鲜**：鲈鱼、带鱼、草鱼、花蛤、蛏子、鱿鱼
**水果**：西瓜、葡萄、桃子、龙眼、无花果、石榴（晚期）
**适合的烹饪方式**：同7月

### 九月（初秋）
**蔬菜**：莲藕、芋头、山药、南瓜、秋葵、西兰花、花菜、茭白
**海鲜**：大闸蟹（孕妇避免）、鲈鱼、鲫鱼、带鱼、虾
**水果**：石榴、柚子、梨、苹果（早期）、柿子（少量，孕期注意）
**适合的烹饪方式**：炖、蒸、煲汤、烤、快炒

### 十月（秋季）
**蔬菜**：莲藕、山药、南瓜、红薯、白萝卜、胡萝卜、西兰花、花菜、茭白
**海鲜**：鲈鱼、鲫鱼、草鱼、虾
**水果**：苹果、梨、柚子、石榴、柿子（少量）
**适合的烹饪方式**：炖、煲汤、蒸、烤

### 十一月（秋末）
**蔬菜**：白萝卜、胡萝卜、土豆、红薯、南瓜、山药、白菜、西兰花、花菜
**海鲜**：带鱼、鲈鱼、黄花鱼、草鱼
**水果**：苹果、梨、橙子、柚子、猕猴桃
**适合的烹饪方式**：炖、煲、蒸、烤、暖锅

### 十二月（冬季）
**蔬菜**：大白菜、白萝卜、胡萝卜、土豆、山药、莲藕、菠菜、芥菜
**海鲜**：带鱼、鲈鱼、黄花鱼、草鱼、青鱼
**水果**：橙子、柑橘、柚子、苹果、梨
**适合的烹饪方式**：炖、煲、蒸、火锅式汤锅

## 四季通用/易购食材

以下食材全年容易购买，不受季节限制：
- **蛋白质**：鸡蛋、鸡胸肉、鸡腿、瘦猪肉、牛肉、豆腐、豆浆、牛奶、酸奶
- **蔬菜**：洋葱、土豆、胡萝卜、豆芽（但芽苗菜建议熟食）、菌菇类（香菇、金针菇、杏鲍菇、口蘑）
- **主食**：大米、小米、糙米、燕麦、面粉、全麦面包、面条、意面
- **调味**：葱、姜、蒜、各种干香料

## 孕期特殊食材的季节性提醒

| 食材 | 最佳季节 | 孕期注意 |
|------|---------|----------|
| 三文鱼 | 进口全年有，秋季最肥 | 确保养殖或太平洋野生，每周≤2-3次 |
| 鳕鱼 | 冬季至初春 | 确保来源，避免冒充鱼种 |
| 菠菜 | 秋冬最甜 | 全年有，焯水去除草酸 |
| 红枣 | 秋季新鲜，干枣全年 | 每天2-3颗，不超量 |
| 核桃 | 秋季新鲜，全年有干核桃 | 每天2-3颗 |
| 黑木耳 | 干木耳全年 | 充分泡发并焯水 |
| 枸杞 | 干枸杞全年 | 每天5-10粒 |
| 牛油果 | 进口全年，夏秋较多 | 每天半个到一个 |
