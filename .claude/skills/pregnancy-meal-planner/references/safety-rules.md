# 孕期食品安全规则

## 绝对禁止的食材与做法

### 生食和未熟食物
| 类别 | 禁止项 | 原因 |
|------|--------|------|
| 肉类 | 生肉、半熟肉、rare/medium-rare 牛排 | 弓形虫、沙门氏菌、李斯特菌感染风险 |
| 禽肉 | 未熟透的鸡肉、鸭肉 | 沙门氏菌、弯曲杆菌 |
| 海鲜 | 生鱼片、刺身、寿司、生蚝、半熟贝类 | 寄生虫、甲肝病毒、李斯特菌 |
| 蛋类 | 生鸡蛋、溏心蛋、温泉蛋、自制蛋黄酱、提拉米苏（含生蛋版） | 沙门氏菌 |
| 豆制品 | 未煮熟的豆浆 | 皂苷、胰蛋白酶抑制剂 |

### 高风险食材
| 类别 | 禁止项 | 原因 |
|------|--------|------|
| 鱼类 | 剑鱼、鲨鱼、鲭鱼（king mackerel）、方头鱼（tilefish）、大眼金枪鱼（bigeye tuna） | 高汞含量，损害胎儿神经系统 |
| 鱼类 | 每月超过2次的深海鱼（如金枪鱼罐头每周不超过1次） | 汞累积风险 |
| 奶制品 | 未经巴氏消毒的牛奶、羊奶 | 李斯特菌、布鲁氏菌 |
| 奶酪 | 软质霉菌成熟奶酪：Brie、Camembert、Blue cheese、Roquefort、Gorgonzola | 李斯特菌高风险（即使巴氏消毒版本也应避免） |
| 肉类 | 冷切肉、生火腿（Prosciutto、Salami、Parma ham） | 李斯特菌（除非加热至 steaming hot） |
| 肉类 | 动物肝脏（猪肝、鸡肝、鹅肝） | 维生素A过量，可能导致胎儿畸形 |
| 肉类 | 未加热的熟食店肉类、卤味凉拌 | 李斯特菌交叉污染 |
| 蔬菜 | 未经彻底清洗的生蔬菜、芽苗菜（苜蓿芽、绿豆芽、萝卜苗） | 沙门氏菌、大肠杆菌（芽苗菜种子本身可能带菌） |
| 饮品 | 酒精类饮品（啤酒、葡萄酒、米酒、料酒未挥发） | 胎儿酒精综合征 |
| 饮品 | 未经医生确认的草药茶、中草药 | 部分草药可能引起宫缩或影响胎儿 |
| 补品 | 未经医生确认的保健品、中药材 | 成分不明确，可能有害 |

### 烹饪禁用方式
- **油炸**：增加不必要的脂肪，产生丙烯酰胺
- **炭烤/明火烧烤**：产生多环芳烃(PAHs)和杂环胺(HCAs)，有致癌风险
- **长时间高温煎炸**：油脂氧化产物对孕妇不利

## 需要严格控制的食物

| 类别 | 限制量 | 说明 |
|------|--------|------|
| 咖啡因 | 每日 ≤ 200mg | ≈ 1杯中杯美式(350ml) 或 2杯速溶咖啡 或 4杯红茶 |
| 含咖啡因饮品 | 咖啡、浓茶、可乐、能量饮料 | 优先建议喝低因咖啡或淡茶 |
| 金枪鱼罐头 | 每周 ≤ 1次（约150g） | 选择 light tuna（skipjack）而非 albacore |
| 三文鱼/沙丁鱼 | 每周 ≤ 2-3次 | 选择养殖或太平洋野生，避免波罗的海三文鱼 |
| 腌制食品 | 尽量不吃 | 咸菜、榨菜、腊肉、咸鱼 — 高盐+亚硝酸盐 |
| 味精/鸡精 | 尽量不用 | 高钠，用天然调味替代 |
| 辣椒 | 适量，有胃灼热时避免 | 可能加重反酸和胃灼热 |
| 山楂 | 避免 | 传统认为可能刺激子宫收缩（孕晚期特别小心） |
| 薏米/薏仁 | 避免 | 传统认为对孕妇不适宜 |
| 螃蟹 | 避免 | 传统认为性寒，且易过敏 |

## 食材处理安全规范

### 肉类处理
1. 烹饪前充分解冻（冰箱冷藏解冻，非室温解冻）
2. 生熟分开：使用不同砧板和刀具
3. 中心温度达到安全值：
   - 猪肉 ≥ 71°C（全熟，无粉色）
   - 鸡肉/鸭肉 ≥ 74°C（全熟，肉汁清澈）
   - 牛肉（如需食用）≥ 71°C（全熟 well-done，孕期不建议五分熟）
   - 鱼肉 ≥ 63°C（肉质不透明，容易剥离）
   - 虾/贝类：壳完全打开/肉质不透明
4. 烹饪后及时食用，室温放置不超过2小时

### 蔬菜水果处理
1. 流动清水充分冲洗（至少30秒）
2. 叶菜类逐片清洗
3. 水果削皮食用更安全（即使苹果、梨这类可带皮的水果）
4. 生食沙拉需格外注意清洗（建议用果蔬清洗剂或盐水浸泡）
5. 芽苗菜建议熟食而非生食

### 蛋类处理
1. 必须全熟：蛋白和蛋黄都完全凝固
2. 水煮蛋：水沸后煮至少8分钟
3. 煎蛋：两面煎至全熟，蛋黄不流动
4. 避免使用破裂的鸡蛋

## 调味料安全清单

### ✅ 孕期安全可用
- 盐（少量）、酱油（控制量）、醋（各种）
- 葱、姜、蒜（新鲜）
- 花椒、胡椒（少量）
- 八角、桂皮、香叶（少量调味用）
- 橄榄油、山茶油、亚麻籽油、牛油果油、菜籽油
- 柠檬汁、番茄
- 迷迭香、百里香、罗勒、莳萝、欧芹
- 芝麻油（少量提香）
- 香菇粉/蘑菇精（天然鲜味替代味精）
- 自制高汤（鸡汤、骨头汤，充分熬煮）

### ⚠️ 需控制的调味料
- 酱油：1汤匙(15ml) ≈ 2g 盐
- 蚝油：1汤匙 ≈ 1g 盐 + 含糖
- 豆瓣酱：高盐高油，少量使用
- 甜面酱：含糖和盐，少量使用
- 辣椒酱：含盐量高，有胃灼热时不用

### ❌ 避免或严格不用
- 味精/鸡精（高钠）
- 料酒（含酒精，即使加热也应避免 — 用姜汁或柠檬汁替代去腥）
- 啤酒/白酒作调味（酒精可能不完全挥发）
- 人造黄油、植脂末（反式脂肪）
- 果葡糖浆、人工甜味剂（如糖精）

## 外卖与外出就餐

如果用户提到外卖或外出就餐，提醒以下注意事项：
- 选择信誉好的餐厅，避免路边摊
- 确认肉类全熟（特别标注 well-done）
- 避免沙拉吧和自助餐（交叉污染风险高）
- 避免现榨果汁（未经巴氏消毒风险）
- 自带餐具更卫生
