# 孕晚期（35周+）营养需求指南

## 核心营养素每日需求

| 营养素 | 每日推荐量 | 作用 | 优质来源 |
|--------|-----------|------|----------|
| 蛋白质 | 70-85g | 胎儿组织生长、子宫和乳腺增大、为分娩和哺乳储备 | 鸡蛋、鸡胸肉、瘦猪肉、牛肉、鱼虾、豆腐、牛奶、酸奶 |
| 钙 | 1000mg | 胎儿骨骼和牙齿钙化（35周后是骨骼发育高峰） | 牛奶(250ml≈300mg)、酸奶、北豆腐、小虾皮、芝麻酱、西兰花、羽衣甘蓝 |
| 铁 | 27mg | 预防分娩贫血、为产后出血做准备、胎儿肝脏储铁 | 瘦牛肉、瘦猪肉、鸭血/猪血、黑木耳、菠菜（配维C促进吸收）、红枣 |
| DHA | 200mg | 胎儿大脑和视网膜发育（35周后是脑发育冲刺期） | 三文鱼、沙丁鱼、鳕鱼、核桃、亚麻籽、奇亚籽 |
| 叶酸 | 600μg | 预防神经管缺陷（虽关键期在孕早期，但全程需要） | 深绿色叶菜（菠菜、西兰花、芦笋）、豆类、牛油果 |
| 膳食纤维 | 25-30g | 预防和缓解便秘（孕晚期常见问题） | 燕麦、糙米、红薯、豆类、蔬菜、水果（带皮） |
| 维生素C | 115mg | 促进铁吸收、增强免疫力、胶原蛋白合成 | 甜椒、西兰花、番茄、猕猴桃、柑橘、草莓 |
| 维生素D | 600IU | 促进钙吸收、免疫调节 | 蛋黄、三文鱼、日晒（每天15-20分钟）、强化奶 |
| 锌 | 11mg | 细胞分裂、免疫支持、为分娩做准备 | 瘦牛肉、瘦猪肉、贝类（确保熟透）、南瓜子 |
| 碘 | 220μg | 胎儿甲状腺功能和脑发育 | 海带、紫菜（少量）、加碘盐、鳕鱼 |
| 水分 | 2-2.5L | 维持羊水量、预防便秘和尿路感染、减轻水肿 | 白开水、淡茶、清汤、含水量高的蔬果 |

## 孕35-40周营养重点变化

### 35-36周：储备期
- 胎儿正在快速储存铁和钙
- 重点：高铁、高钙、优质蛋白
- 此时胎儿体重约2.5-2.7kg，每周增重约200-250g
- 孕妇血容量达到峰值，需要更多铁质防贫血

### 37-38周：成熟期
- 胎儿肺部和其他器官最后成熟阶段
- 重点：DHA（脑发育冲刺）、蛋白质、维生素K（凝血功能）
- 此时胎儿体重约2.9-3.2kg
- 孕妇可能感到胃部受压减轻（胎儿入盆），食量可适当增加
- 少食多餐比三大餐更舒适

### 39-40周：待产期
- 为分娩储备能量
- 重点：易消化的高能量食物、碳水化合物（为产程提供能量）、铁（防分娩出血）
- 避免产前大补（胎儿过大不利顺产）
- 保持水分充足
- 可适当增加进食次数，每餐少量

## 常见身体状况的饮食调整

### 水肿
- **核心策略**：控盐 + 增加利尿食材
- **推荐食材**：冬瓜、黄瓜、芹菜、红豆、薏米（跳过，孕期不宜）、玉米须茶、西瓜（适量）
- **增加钾摄入**：香蕉、土豆、番茄、菠菜、牛油果（钾钠平衡有助于排水）
- **盐严格控制在每日 < 3g**
- **避免**：咸菜、腌制食品、酱油过量、加工食品

### 妊娠期糖尿病 (GDM)
- **核心策略**：控制血糖波动 + 碳水管理
- **推荐主食（低GI）**：糙米、燕麦、荞麦、全麦面包、杂豆饭、红薯（少量）
- **避免**：白米饭、白面包、白面条、糯米、甜点、含糖饮料、果汁
- **进餐顺序**：汤 → 蔬菜 → 蛋白质 → 主食（最后吃碳水，减缓血糖上升）
- **水果选择**（低GI）：蓝莓、草莓、樱桃、柚子、苹果（每天不超过200g，分2次）
- **加餐**：建议每日分为5-6餐（3正餐 + 2-3加餐），避免单次大量进食
- **监测**：如用户有血糖仪，建议饭后2小时测血糖

### 宫高偏大/胎儿偏大（LGA倾向）
- **核心策略**：调整营养结构而非简单节食——减少精制碳水和隐藏脂肪，增加优质蛋白比例和膳食纤维
- **绝对不是节食！**胎儿偏大不等于要让孕妇少吃，而是改变吃什么、怎么吃
- **碳水管理**（不需要像GDM那么严格，但要重视）：
  - 优先低GI主食：燕麦、荞麦、糙米、玉米、南瓜、红薯（中等量）
  - 减少白米饭比例，改为杂粮饭（白米:杂粮 = 2:1 或 1:1）
  - 晚餐主食量比午餐减少1/3
  - 避免：糯米饭、炒饭、炒面、面包、糕点
- **蛋白质**：保证每餐有优质蛋白（鱼、去皮鸡肉、豆腐、蛋），但不过量——每人每餐肉类100-120g（生重）即可
- **蔬菜翻倍**：每餐蔬菜量增加到餐盘的1/2以上，先吃蔬菜再吃蛋白最后吃主食
- **隐藏脂肪控制**：
  - 肉类去皮去肥、汤品必须撇油
  - 不用油炒青菜时改为"水油炒"（水+3ml油）
  - 坚果每天限量（核桃2颗或杏仁10粒，不要当零食随便吃）
- **水果控制**：每天200g以内，选择低GI的（苹果、柚子、蓝莓、草莓），避免西瓜、荔枝、龙眼、芒果
- **进食速度**：细嚼慢咽，每餐吃20分钟以上，给大脑饱腹信号时间
- **适量运动**：饭后慢走15-20分钟（和GDM管理一样重要）
- **需要警惕的情况**：如果同时有GDM——按GDM方案严格执行；如果同时有高血压——联合管理更需严格控制
- **心态**：宫高偏大一周左右很常见，多数情况下通过饮食调整可以控制，不要焦虑

### 妊娠期高血压
- **核心策略**：极严格控盐 + 控油 + 优质蛋白
- **盐**：每日 < 2g（极严格）
- **推荐食材**：芹菜、苦瓜、洋葱、大蒜、海带、木耳、香蕉（富钾）
- **优质蛋白**：鱼肉（每周2-3次）、去皮鸡肉、豆腐
- **避免**：红肉肥腻部分、动物内脏、油炸食品、咸味调料
- **烹饪方式**：仅蒸、煮、炖，不炒

### 便秘
- **核心策略**：增加纤维 + 充足水分 + 适当油脂润肠
- **高纤维食材**：燕麦、红薯、玉米、芹菜、韭菜、西兰花、火龙果、西梅、奇亚籽
- **水分**：晨起一杯温水，全天不少于2L
- **润肠**：橄榄油（少量）、芝麻、核桃、酸奶（益生菌）
- **避免**：精细米面、辛辣食物、过多红肉

### 胃灼热/反酸
- **核心策略**：少食多餐 + 避免诱因
- **细嚼慢咽**，每餐七八分饱
- **避免**：辛辣、油炸、过酸（柠檬、醋过量）、巧克力、薄荷、咖啡、碳酸饮料
- **饭后保持直立**至少1-2小时，不要立即躺下
- **晚餐**尽量在睡前3小时完成
- **适合**：燕麦、香蕉、蒸蔬菜、土豆泥、瘦肉、鱼类

### 贫血（缺铁性）
- **核心策略**：高铁 + 维C促进吸收
- **血红素铁**（吸收率高）：瘦牛肉、瘦猪肉、鸭血、猪血、鸡肝（肝类孕晚期慎用）
- **非血红素铁**：菠菜、黑木耳、红枣、黑芝麻
- **铁吸收促进**：搭配维C丰富的食物（甜椒、番茄、西兰花、柠檬汁）
- **铁吸收抑制**：避免与浓茶、咖啡、牛奶同时食用（错开至少1小时）
- **推荐搭配**：菠菜炒猪肝（猪肝每周1次且不超过30g）+ 挤柠檬汁

### 睡眠不佳
- **晚餐推荐**：富含钙镁（牛奶、酸奶）、色氨酸（火鸡/鸡肉、香蕉、坚果）、复合碳水（全麦面包、燕麦）
- **睡前1小时**：一杯温牛奶 + 少量全麦饼干
- **避免**：晚餐过饱、咖啡因、巧克力、辛辣食物
- **白天**：适量户外活动，日光帮助调节褪黑素

## 每日能量需求

| 孕期 | 额外热量（基于孕前） |
|------|---------------------|
| 孕早期（1-12周） | +0-100 kcal |
| 孕中期（13-28周） | +300-350 kcal |
| 孕晚期（28-40周） | +450 kcal |

以孕前1800kcal为例，孕晚期约需2250kcal/天。
孕晚期最后几周（37-40周），如果活动减少，可适当降低至+300kcal。

## 餐次分配建议（以2250kcal为例）

| 餐次 | 热量占比 | 大约热量 |
|------|---------|---------|
| 早餐 | 25% | ~560 kcal |
| 上午加餐 | 10% | ~225 kcal |
| 午餐 | 30% | ~675 kcal |
| 下午加餐 | 10% | ~225 kcal |
| 晚餐 | 20% | ~450 kcal |
| 睡前加餐 | 5% | ~115 kcal |

晚餐热量较低有助于睡眠，午餐可以吃得丰盛一些。
